# Barber Beats

Un procesador de audio web inspirado en el género musical [Barber Beats](https://rateyourmusic.com/genre/barber-beats/#:~:text=1%2C610%20releases-,Barber%20beats,-primarily%20developed%20from).

Permite carga de audios, aplicar efectos en tiempo real, visualizar espectros y formas de onda, y exportar la mezcla final.


## Quick Start

1) **Node.js** (< 18)
2) **pnpm** (`npm` / `yarn`)


```bash
git clone https://github.com/1toe/barberbeats
cd barberbeats
```

```bash
pnpm install
```
```bash
pnpm dev
```

3) Dev @ `http://localhost:3000`
